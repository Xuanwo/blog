CodeVS 1212 最大公约数

2014-10-15 / Comments

题目

源地址：

http://codevs.cn/problem/1212/

理解

水题。 只要使用gcd递归就可以搞定。

新技能get

GCD模板代码

int gcd(int a, int b)
{
	return b==0?a:gcd(b,a%b);
}

代码


#include <cstdio>
#include <cstdlib>
#include <cstring>

using namespace std;

ll a,b;

ll gcd(ll a, ll b)
{
	return b==0?a:gcd(b,a%b);
}

int main(int argc, char const *argv[])
{
	scanf("%lld%lld", &a, &b);
	printf("%lld", gcd(a,b));
	return 0;
}
