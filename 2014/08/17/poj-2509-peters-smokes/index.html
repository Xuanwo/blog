POJ 2509 Peter's smokes

2014-08-17 / Comments

题目

源地址：

http://poj.org/problem?id=2509

理解

模拟每个人吸烟的过程即可。

代码

#define inf (1<<30)
using namespace std;


int main(int argc, char const *argv[])
{
    int n;
    int k;
    int sum = 0;
    while (cin >> n >> k)
    {
        sum = n;
        while (n / k)
        {
            sum += (n / k);
            n = n % k + n / k;
        }
        cout << sum << endl;
    }
    return 0;
}
