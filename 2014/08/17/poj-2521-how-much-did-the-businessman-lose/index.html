POJ 2521 How much did the businessman lose

2014-08-17 / Comments

题目

源地址：

http://poj.org/problem?id=2521

理解

小学的时候经常做的题目啊，只要保持头脑清醒，计算出交易过程中因为假币损失的钱。

代码

#include <cstdlib>

#include <fstream>

#define eps (1e-8)
#define inf (1<<30)
using namespace std;

int main(int argc, char const *argv[])
{
    int n, m, p, c;
    while (cin >> n >> m >> p >> c && n + m + p + c)
    {
        cout << p - (m - n) << endl;
    }

}
