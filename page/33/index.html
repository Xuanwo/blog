POJ 1458 Common Subsequence

2014-08-12
Code

题目

源地址：

http://poj.org/problem?id=1458

理解

DP的基础题，求最长子序列（LCS）。 状态转移方程伪代码如下：

if (i == 0 || j == 0)
	dp[i,j] = 0
else if (X[i] == Y[j])
	dp[i,j] = dp[i-1,j-1] + 1
else
	dp[i,j] = max(dp[i-1,j], dp[i,j-1])

POJ 1159 Palindrome

2014-08-12
Code

题目

源地址：

http://poj.org/problem?id=1159

理解

最关键的公式是：

if (str[i] == str[j])
    DP[k][j] = DP[1 - k][j - 1];
else
    DP[k][j] = min(DP[1 - k][j], DP[k][j - 1]) + 1;

注意到k空间的循环利用，节省了空间，感谢大牛提供的思路。
