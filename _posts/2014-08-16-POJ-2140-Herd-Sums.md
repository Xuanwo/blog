# 理解
很有意思的一道题，很显然，i不会超过n/2。
如果是奇数，我只要考虑最中间的两个数；如果是奇数，我只要在上面计算的结果上再加1。

    int i, n;
    int counts = 0;
    scanf("%d", &n);
    for (i = 1; i <= n / 2; i += 2)
        if (n % i == 0)
            counts++;
    if (n % 2 != 0)
        counts++;
    printf("%d\n", counts);
    return 0;
}
