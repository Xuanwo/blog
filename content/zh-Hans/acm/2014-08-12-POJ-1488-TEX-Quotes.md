## 题目
源地址：

http://poj.org/problem?id=1488

# 理解
字符串处理- -，慢慢坑就是了。。注意输入方式，有空格的时候要用getline。

<!--more-->

# 代码

```
#include<iostream>
#include<string>
using namespace std;

int main(int argc, char const *argv[])
{
    string a ;
    int count = 0;
    while (getline(cin, a))
    {
        for (int i = 0; i < a.length(); i++)
        {
            if (a[i] == '\"')
            {
                if (count == 0)
                {
                    cout << "``";
                    count = 1;
                }
                else
                {
                    count = 0;
                    cout << "''";
                }
            }
            else cout << a[i];
        }
        cout << endl;
    }
}

```
