## 题目
源地址：

http://poj.org/problem?id=1401

# 理解
一开始惊讶地发现，每个数除以4之后都跟答案比较接近，然后打算开始偷鸡，结果失败了= =。实际上，是每一个数不停地除以5，寻找10的因子的过程。

<!--more-->

# 代码

```
#include <iostream>

using namespace std;

int main()
{
    int T;
    cin >> T;
    long long int  N;
    for (int i = 0; i < T; i++)
    {
        cin >> N;
        int ret = 0;
        while (N)
        {
            ret = ret + N / 5;
            N = N / 5;
        }
        cout << ret << endl;
    }
    return 0;
}

```
