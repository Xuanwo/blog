## 题目
源地址：

http://poj.org/problem?id=2390

# 理解
给定年利率，本金和存款年数，求解到期之后的本息和。找自信专用。

# 代码

```

#include <cstdio>
#include <cstdlib>

#include <fstream>

#define eps (1e-8)
#define inf (1<<30)
using namespace std;

int main(void)
{
    int r, m, y;
    int i;
    double value, rate;
    while (scanf("%d%d%d", &r, &m , &y) != EOF)
    {
        rate = 1 + (double)r / 100.0;
        value = m;
        for (i = 0; i < y; i++)
        {
            value *= rate;
        }
        printf("%d\n", (int)value);
    }
    return 0;
}
```
