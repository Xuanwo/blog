## 题目
源地址：

http://poj.org/problem?id=1936

# 理解
字符串的题目。因为数据水所以这是一道水题，要是数据再强一点估计就TLE了。

<!--more-->

# 代码

```
#include <stdio.h>
#include <string.h>
#define N 100001
using namespace std;

char s[N], t[N];
int slen, tlen, i, j, tag;

int main()
{
    while ( scanf("%s  %s", s, t) != EOF )
    {
        slen = strlen(s);
        tlen = strlen(t);
        for (i = 0 , j = 0; i < tlen && j < slen; ++i)
            if (s[j] == t[i])
                ++j;
        if (j == slen)  puts("Yes");
        else puts("No");
    }
    return 0;
}

```
