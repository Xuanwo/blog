# 理解
额，就是求解12个数的平均数。不过据说POJ不能用lf，还要我没有遇到这样的问题，交了一个float水过。

<!--more-->

# 代码

```
#include <stdio.h>
#include <iostream>

using namespace std;

int main()
{
	int n=12;
	float money,sum=0;
	for(int i=0;i<12;i++)
		{
			cin>>money;
			sum+=money;
		}
	printf("$%.2f", sum/12);
	return 0;
}

```
