## 题目
源地址：

http://codeforces.com/contest/9/problem/A

# 理解
阅读题= =，实际上，只要Dot的点数比其他两个人中大的那个人大就行。
直接用switch搞定。

<!--more-->

# 代码

```

int m,n;

int main(int argc, char const *argv[])
{
    while(~scanf("%d%d",&n,&m))
    {
        if(n<m)
            swap(n,m);
        int x=6-n+1;
        switch(x)
        {
        case 0:
            printf("0/1\n");
            break;
        case 1:
            printf("1/6\n");
            break;
        case 2:
            printf("1/3\n");
            break;
        case 3:
            printf("1/2\n");
            break;
        case 4:
            printf("2/3\n");
            break;
        case 5:
            printf("5/6\n");
            break;
        default:
            printf("1/1\n");
        }
        return 0;
    }
}


```
