## My Blog

This repo hosts https://xuanwo.io.

## Development

- `npm install`
- `npm run dev`

## Build

- `npm run build`

## Sketches

- Put self-contained HTML bundles under `static/sketches/<slug>/index.html`.
- Add list metadata to `src/data/sketches.ts`.
- The collection index is `/sketches/`.
- Sketch links point to `/sketches/<slug>/index.html`.

## Configuration

- `site.yaml`: site metadata and navigation
